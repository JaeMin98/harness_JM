# Harness Core Index

## 1. Purpose
이 문서는 repository-level harness 문서의 진입점이다.
먼저 이 문서를 읽고, 필요한 하위 문서로 이동한다.

## 2. Read Order
1. `constitution.md`: 최상위 원칙
2. `core-values.md`: 핵심 가치 요약
3. `governance.md`: 문서 경계와 변경 규칙
4. `repository-architecture.md`: 저장소 구조
5. `repository-security.md`: 공통 보안 기준
6. `repository-reliability.md`: 공통 검증 기준

## 3. Use Rule
- 공통 규칙은 여기서 찾는다.
- 앱별 규칙은 각 앱의 `apps/<app-name>/harness/` 아래에서 찾는다.
- 문서가 충돌하면 `constitution.md`를 먼저 따른다.
