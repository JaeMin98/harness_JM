# Tracker Template

## Purpose
앱 전체 진행 상태를 한눈에 남기는 템플릿이다.

## Template
```md
# Tracker

## Current Phase
- phase:
- status:

## Current Step
- owner:
- state:

## Current Work
- title:
- path:

## Verification
- verdict:
- attempts:

## Next Owner
- owner:

## Next Action
- next:

## Issues
- none
```

## Rule
- 현재 상태만 남긴다.
- 상세 작업 내용은 `ongoing/` 문서에 둔다.
- 단계가 바뀌면 바로 갱신한다.
- 판정 값과 시도 횟수는 비워 두지 않는다.
